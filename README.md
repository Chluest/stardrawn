A collaborative random entry picker with real-time sync.

## Features
- Create a shared room and invite friends via URL
- Add and remove entries in real time
- Pick a random entry with a dramatic reveal
- Solo mode for quick private picking

## Tech Stack
**Frontend:** React, Vite
**Backend:** Python, FastAPI, WebSockets
**Database:** PostgreSQL
**Real-time:** WebSockets with Redis pub/sub (coming soon)
